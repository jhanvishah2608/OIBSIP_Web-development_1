# OIBSIP_Web-development_1

--Objective:
To develop a simple web-based calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

--Steps Performed:

1. Created the calculator layout using HTML.
2. Added buttons and input fields for numbers and operations.
3. Used JavaScript to perform arithmetic calculations.
4. Displayed the calculated result on the webpage.

--Tools Used:

* HTML
* CSS
* JavaScript
* VS Code / Any Web Browser

--Outcome:
The calculator application successfully performs basic mathematical operations through a user-friendly web interface and displays accurate results instantly.
